# Portfolio Website v2

A modern personal portfolio website built with Next.js, focused on polished UI design, smooth animations, and interactive user experiences.

## About

This is the second iteration of my personal portfolio website and will continue to evolve as I build new projects and develop new skills.

The goal of this portfolio was to create a more immersive and professional experience compared to the first version, with a stronger emphasis on visual design, motion, responsiveness, and overall presentation.

The website includes features such as:
- Interactive project showcases
- Animated page transitions
- Responsive layouts across devices
- Draggable and interactive skills section
- Project carousels and polished UI components

While the site is primarily frontend-focused, it serves as a platform to showcase my projects, technical skills, and approach to building modern web applications.

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Framer Motion
